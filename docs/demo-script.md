# Demo script (3–5 phút)

1. Chạy seed, backend và frontend.
2. Chỉ runtime badge `FPT_LLM · FPT_VECTOR_RAG` (hoặc fallback nếu chủ động demo offline).
3. Tạo một case mới; tải application, contract, payslip và statement.
4. Chạy pipeline; giải thích Income/Policy chạy song song.
5. Mở evidence Document Agent và citation Policy Agent.
6. Trình bày average/eligible income, calculation version, findings và critic summary.
7. Reload trang, mở lại case để chứng minh SQLite persistence.
8. Mở case thiếu statement để thấy `AWAITING_DOCUMENTS` thay vì hallucination.
9. Review action và nhấn mạnh mock adapters + human approval; hệ thống không phê duyệt khoản vay.
